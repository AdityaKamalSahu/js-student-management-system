# Student Management System

## Description

This is my first JavaScript mini project. I built it to practice the concepts from JavaScript Module 2 such as switch, if-else, logical operators, and basic calculations.

## Features

- Show student details
- Calculate total marks
- Calculate percentage
- Check pass or fail
- Display grade
- Check scholarship eligibility

## Technologies Used

- JavaScript
- Node.js
- VS Code

## Concepts Used

- Variables
- Arithmetic operators
- if, else if, else
- Logical operators (&&)
- switch-case
- break
- default

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
node index.js
```

## About

This project was created as part of my JavaScript learning journey to improve my programming logic before moving on to loops and functions.
